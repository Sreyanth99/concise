# Concise
A website that makes lengthy articles more concise!\
Works well on Mobile and Desktop

Features:

-Article Summary\
-Article History\
-Slick UI\
-OpenAI API\
-Copy To Clipboard\
-Error Handling\
-Animations

Technologies Used:

-Vite\
-React\
-TailwindCSS\
-RTK


Head over to **[Concise-App](https://concise-srey-i.netlify.app/)** to try out the live version
