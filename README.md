# Concise
A website that makes lengthy articles more concise!\
Works well on Mobile and Desktop

<b><i>Features:</i></b> \
-Article Summary\
-Article History\
-Slick UI\
-OpenAI API\
-Copy To Clipboard\
-Error Handling\
-Animations

<b><i>Technologies Used:</i></b> \
-Vite\
-React\
-TailwindCSS\
-RTK
